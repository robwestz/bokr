# 04_SKILLS_AND_GATES — Skills + quality gates (agnostic)

## Core skills
- Requirements distillation (mätbara krav + acceptance)
- Architecture/contracts (komponenter + data/API)
- Task decomposition (owner, I/O, DoD, deps)
- Test mapping (smoke/unit/integration/e2e/security)
- Agent continuity (macro brief + checkpoints + merge rules)

## Gates (måste PASS)
G1 Input sufficiency  
G2 Completeness vs MINIMAX (alla ID täckta)  
G3 Executability (tasks har owner/I/O/DoD/deps)  
G4 Security baseline (RBAC/isolation/audit/secrets/uploads)  
G5 Testability (smoke+e2e+unit+integration+security)  
G6 Ops readiness (runbook build/test/deploy/rollback + observability basics)
